## Just a chill bot
----------

This bot can filter links, files, swears from blacklists

It can also do basics moderations commands for ban, unban, pardon, warn, and so on.

It also can clear messages of an user in a room or for the whole server. with "Nettoyer" for once room and "Purge" for the whole messages of the user in the whole server.

The commands for the operators (developpers) are also availables for reboot, update, backup or as well register a new server.
